Uncal

Self-hosted calendar booking for small teams. Like Calendly, but you own it.

Built with Rails 8, SQLite, and Hotwire. No Redis, no Node.js, no external services required.

Everything you need, nothing you don't

🔒

Invite-only teams

No public sign-up. The first admin is seeded via a rake task, everyone else joins through email invitations.

🔗

Schedule links

Create booking pages with custom slugs. Add multiple team members to a single link for group availability.

🕓

Availability windows

Define per-member weekly availability with a visual grid. Copy schedules between links in one click.

📅

Google Calendar sync

Connect Google Calendar to automatically check for conflicts. OAuth flow with automatic token refresh.

🌍

Timezone-aware booking

Visitors see available slots in their local timezone. All times stored in UTC, converted on display.

✉

Email workflows

Automated confirmation, reminder, and follow-up emails. Build custom multi-step workflows with markdown templates.

📋

Contacts & CRM-lite

Every booker becomes a contact. Track booking history, notes, and communication in one place.

📊

Dashboard with stats

See upcoming bookings, recent activity, and key metrics at a glance from a clean overview page.

✅

No external dependencies

SQLite for everything — database, queue, cache, and WebSockets. No Redis, Postgres, or background services to manage.

Built with

Rails 8 SQLite Hotwire Tailwind CSS Solid Queue Solid Cache Solid Cable Kamal Propshaft

Get started in minutes

1

Clone the repository

git clone https://github.com/Cause-of-a-Kind/uncal.git

2

Install dependencies and set up the database

bin/setup

3

Create the first admin user

bin/rails setup:admin

4

Start the server

bin/rails server
